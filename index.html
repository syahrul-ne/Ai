Tentang AI Chatbot

AI Chatbot adalah asisten virtual cerdas yang dapat membantu Anda dalam berbagai hal. Aplikasi ini dibuat menggunakan HTML, CSS, dan JavaScript murni, sehingga dapat dijalankan langsung di browser tanpa memerlukan server.

Fitur:

  • Multiple AI personality modes
  • Responsive design untuk semua perangkat
  • Export chat history
  • Voice input simulation
  • Real-time typing indicator
  • Optimized untuk Android dan iOS
×

Bantuan

Cara Menggunakan:

  1. Pilih mode AI yang diinginkan dari dropdown
  2. Ketik pesan Anda di kolom input
  3. Tekan Enter atau klik tombol kirim
  4. AI akan merespons sesuai dengan mode yang dipilih

Tips Mobile:

  • Gunakan orientasi portrait untuk pengalaman terbaik
  • Area chat memiliki scroll sendiri untuk navigasi mudah
  • Semua tombol sudah dioptimalkan untuk sentuhan
  • Input field mencegah zoom otomatis di iOS

Tips Umum:

  • Gunakan mode "Profesional" untuk pertanyaan bisnis
  • Mode "Kreatif" cocok untuk brainstorming
  • Mode "Teknis" untuk pertanyaan programming
  • Klik "Export" untuk menyimpan percakapan
